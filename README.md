# MyCPP
 Simple C++ Game with Raylib
 
 Installation: Just Build and Run

**Note:** You need to download the dependencies in order to build the project

**Dependencies:** [Raylib](https://www.raylib.com/)
 
 Features:
 - 2D Top Down
 - Basic Movement
 - Simple AI that chases player
 - Health System
 - Collision System
 
 Controls:

- Movement = W-A-S-D
- Attack = Left Click
